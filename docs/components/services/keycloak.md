# Keycloak

[Keycloak](https://keycloak.org) is the IdP (**Id**entity **P**rovider) I use
for my services. It provides both
[OpenID Connect](https://en.wikipedia.org/wiki/OpenID_Connect) and
[SAML](https://en.wikipedia.org/wiki/Security_Assertion_Markup_Language)
protocols for me to sign in to the other services in my cluster with.
